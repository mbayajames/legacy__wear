# Legacy Wear

![Legacy Wear Homepage Screenshot](https://htmlburger.com/blog/wp-content/uploads/2024/06/modern-dark-theme-websites.jpg)

**Elevate Your Style Legacy** — Premium online fashion retailer offering elegant clothing, stunning heels, statement accessories, and more.

Live Demo: [https://legacy-wear.vercel.app/](https://legacy-wear.vercel.app/)

## Overview

Legacy Wear is a modern e-commerce platform specializing in high-quality, elegant fashion items. Discover premium collections of footwear (heels, sneakers, pumps), clothing (dresses, blazers), and accessories (tote bags, clutches, watches) designed to enhance your personal style.

Built with a clean, user-friendly interface emphasizing trust signals like free delivery, secure payments, and easy returns — perfect for fashion lovers in Kenya and beyond.

## Key Features

- **Elegant Hero Section** — "Elevate Your Style Legacy" with premium product showcases
- **Shop by Category** — Easy navigation to curated collections (shoes, clothes, etc.)
- **Featured & Sale Items** — Handpicked new arrivals, discounts (e.g. Pink Friday Sale up to 30% off), product ratings, and quick add-to-cart
- **Trust & Convenience**  
  - Free delivery on orders above KSh 10,000  
  - Secure payments (M-Pesa, cards, bank transfers)  
  - 7-day hassle-free returns
- **Customer Testimonials** — Real feedback highlighting exceptional quality
- **Responsive Design** — Looks great on mobile, tablet, and desktop
- **Prominent CTAs** — "Start Shopping" drives users to collections

## Tech Stack (Inferred from Deployment & Structure)

- **Frontend Framework**: Next.js (or React)  
- **Deployment**: Vercel (automatic previews, global CDN)  
- **Styling**: Tailwind CSS or similar modern utility-first approach  
- **Payments**: Integration-ready for M-Pesa, cards, etc. (frontend-focused)  
- **Other**: Likely includes cart state management (Zustand / Redux / Context), image optimization via Next.js
