## ✨ About The Project

**Legacy Wear** is a premium e-commerce platform for curated fashion. It showcases high-end products with a focus on elegant design, seamless user experience, and mobile-first responsiveness.

### 🚀 Key Features

*   **Curated Collections:** Browse handpicked items like *Midnight Rose Heels*, *Noir Elegance Dress*, and *Rose Gold Watch*.
*   **Prominent Sales:** Dynamic "Sale" and "NEW" tags with clear original vs. discounted pricing.
*   **Customer Trust:** Integrated 5-star reviews, ratings, and testimonials from verified buyers.
*   **Service Highlights:** Clear display of Free Delivery, Secure Payment, and Easy Returns policies.
*   **Call-to-Action:** Strategic "Pink Friday Sale" banner and newsletter signup section.
